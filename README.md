## Notes Included

### Linux Commands

Documents basic Linux commands for navigation, file management, system checks, networking, processes, and troubleshooting.

### Troubleshooting Notes

Documents a basic troubleshooting process for identifying issues, checking the basics, testing one change at a time, and recording results.

### Network Troubleshooting Checklist

Covers basic steps for checking network connectivity, IP address information, internet access, and DNS/name resolution.

### Linux Permissions

Documents permission basics, including read, write, execute, ownership, and common `chmod` examples.

### Package Management

Documents common package management tasks such as updating, upgrading, searching, installing, and removing software.

### Service Management

Documents basic service checks using `systemctl`, including how to check service status and list services.

### Log File Practice

Documents basic Linux log review using `/var/log`, `journalctl`, and `dmesg`.

### Linux Cheat Sheet

Provides a quick reference for common Linux commands used in troubleshooting and system support.

### Bash Scripting

Documents a beginner Bash script used to collect system information for troubleshooting.

### Python Scripting

Documents a beginner Python script used to collect basic issue information and create a ticket-style summary.

## Scripts Included

### system-info.sh

A Bash script that gathers basic Linux system information, including:

* Hostname
* Current user
* Kernel version
* Disk usage
* Memory usage
* Network interface information
* Recent system logs

### hello-it-support.py

A beginner Python script that collects basic troubleshooting information and prints a ticket-style summary.
