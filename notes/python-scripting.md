This file documents beginner Python scripting practice from my Linux home lab.

## Purpose

The goal of this lab is to practice basic Python scripting in a way that connects to IT support and troubleshooting documentation.

## Script Created

File:

```text
scripts/hello-it-support.py
```

## What the Script Does

The script asks for:

* Device name
* Issue description
* Troubleshooting steps taken

Then it prints a basic ticket-style summary.

## Skills Practiced

* Running a Python script
* Using `print()` statements
* Using `input()` to collect information
* Storing information in variables
* Creating a simple support-style summary
* Connecting scripting practice to IT support workflows

## Why This Matters for IT Support

Even simple scripts can help with documentation, automation, and repeatable workflows. This script is beginner-level, but it helps me practice thinking like an IT support technician by gathering issue details and organizing them clearly.
