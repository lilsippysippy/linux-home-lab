# Linux Commands Practice

This file documents Linux commands I am practicing in my home lab.

## Navigation Commands

### pwd

Shows the current directory I am working in.

### ls

Lists files and folders in the current directory.

### cd

Changes directories.

Example:

```bash
cd Documents
```

## File and Folder Commands

### mkdir

Creates a new folder.

Example:

```bash
mkdir test-folder
```

### touch

Creates a new empty file.

Example:

```bash
touch notes.txt
```

### cp

Copies a file or folder.

### mv

Moves or renames a file or folder.

### rm

Removes a file.

## System Update Commands

### sudo apt update

Checks for available package updates.

### sudo apt upgrade

Installs available updates.

## Networking Commands

### ip a

Shows network interface and IP address information.

### ping

Tests network connectivity.

Example:

```bash
ping google.com
```

## Process Commands

### ps

Shows running processes.

### kill

Stops a process by process ID.

## Notes
I am continuing to practice these commands to build confidence with Linux troubleshooting, system navigation, and command-line support.
